# client

React + TypeScript + Vite client package.

## Run from repository root

Start dev server:

```bash
bun run client
```

Build:

```bash
bun run client:build
```

Preview build:

```bash
bun run client:preview
```

## Run from packages/client

Start dev server:

```bash
bun run dev
```

Build:

```bash
bun run build
```

Preview:

```bash
bun run preview
```

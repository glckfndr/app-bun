# server

Express server package.

## Environment

Create packages/server/.env and set values like:

```env
OPENAI_API_KEY=your_key_here
PORT=3000
```

## Run from repository root

```bash
bun run server
```

Watch mode from root:

```bash
bun run dev
```

## Run from packages/server

```bash
bun run server
```

Watch mode locally:

```bash
bun run dev
```
